Projet : Interpréteur SimpleText → HTML
=======================================

Pour compiler et exécuter le projet :

1. Compilation (avec dune recommandé) :
   dune build

2. Exécution :
   dune exec ./main.exe <exemple.stxt>

Fichiers importants :
- ast.ml : définition de l’AST
- lexer.mll : analyseur lexical
- parser.mly : analyseur syntaxique
- interpreteur.ml : génération du HTML
- main.ml : point d’entrée du programme

Exemple d’entrée : voir le dossier exemples/
